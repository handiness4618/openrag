A Docker or Podman VM must be running before you start OpenRAG unless you are running OpenRAG on a Linux-based VM, such as a Linux-based [WSL](/install-windows) image.

The container VM must have sufficient resources to run the OpenRAG containers.
The minimum recommendation is 8 GB of RAM and at least 50 GB of free disk space.

For more information, see [Troubleshoot OpenRAG](/support/troubleshoot).
